EduTab OneLink

Jedna HTML aplikace pro žáky i učitele. Žák vidí jen jednoduché ovládání tabulky. Učitel může skrytě otevřít admin režim, vytvořit zadání, vygenerovat sdílecí odkaz a nastavit odesílání výsledků do Google Sheets přes Google Apps Script.
Admin režim aktivní. Navenek zůstává jeden odkaz, ale učitel má navíc tvorbu zadání, sdílecí link a nastavení Google Sheets.

3. Google Sheets
Aplikace odešle JSON přes POST. Apps Script zapíše řádek do tabulky a vrátí potvrzení s pořadovým číslem odevzdání. Pokud URL není vyplněné, výsledek se stále dá exportovat jako JSON.

Krok po kroku
1. Načti zadání nebo otevři sdílecí odkaz od učitele.
2. Spusť školní režim a plň kroky ve správném pořadí.
3. Ukládej průběžně a po dokončení odešli výsledek.
4. Když je nastavený Google Sheets, stačí kliknout na odeslání.
Technická poznámka: zapisování do Google Sheets funguje přes Google Apps Script Web App URL. Tato aplikace je připravená na přímé odeslání dat, ale je potřeba jednorázově vyplnit a nasadit Apps Script.
