AI generování zadání

Důležité: Tato verze umí zadání generovat dvěma způsoby: 1) lokální chytrý generátor bez API a 2) přes AI endpoint, pokud si ho učitel nastaví přes Apps Script nebo jiný backend. API klíč nepatří do HTML stránky.

Jak AI režim funguje

Tlačítko AI návrh bez API vytvoří smysluplné zadání přímo v prohlížeči. Je vhodné pro rychlou přípravu hodin a funguje i bez internetu navíc.
Tlačítko Generovat přes AI endpoint odešle parametry na tebou zadaný backend a očekává JSON odpověď. To je správná cesta, pokud chceš skutečné AI generování přímo uvnitř adminu.
Tlačítko Vytvořit prompt ti připraví přesný prompt, který můžeš poslat do ChatGPT. Odpověď ve formátu JSON pak vložíš zpět přes Použít vložený AI JSON.

Distribuce žákům

Žákovský odkaz neotevírá admin automaticky. Žák uvidí jen zadání a pracovní část.
